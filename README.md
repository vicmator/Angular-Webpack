# Angular2-Webpack
It's a single page using Angular2 and Typescript with Webpack

# How to get started

Clone or download this project.

From the root folder of the project execute (command prompt):

```
npm install
```

Then

```
npm start
```

Open a browser and navigate to http://localhost:8080
